使用单选框， 复选框以供用户选择


  1. 语法 %input type="radio/checkbox" value="值" name="名称" checked="checked""

  2. type = radio 控件为单选框 type="checkbox" 控件为复选框
  3. value:提交数据到服务器的值（后台程序PHP使用）
  4. name:为控件命名，以备后台程序ASP,PHP使用
  5. checked:当设置checked="checked"时，该选项被默认选中
